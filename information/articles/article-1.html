Pulsar un elemento para accionar otros elementos

Para accionar algún efecto, animación, cambio... en un elemento B (o varios) al pulsar (hover, focus,...) un elemento X, haciendo uso exclusivamente de CSS y HTML, se debe hacer uso de los selectores de CSS, como *, X > Y, W Y, P ~ D...

Ejemplo:

A 
      
        <div class="parent">
        <span class="elemento-1">A</span>
        <span class="elemento-2"></span>
        <span class="elemento-3"></span>
        </div>

Tenemos un parent con position: relative y unos 3 hijos con position: absolute, lo cual permite que puedan estar uno por encima del otro. Por supuesto, usando la propiedad z-index, la cual permite al elemento con mayor número, estar por encima de los otros y así sucesivamente.

Básicamente necesitamos un elemento accionador, que al ser pulsado, usando :hover u otra pseudo clase como :focus, active el efecto sobre los otros elemento o el otro elemento. Pero siempre haciendo uso de selectores como A > B, A B, A ~ B, A + B...

En este caso se usa las propiedades width y height, para hacer crecer el tamaño de los otros elementos (2 y 3) cuando el elemento 1 (accionador) sea activado mediante :hover

Además se agrega un efecto más natural con la propiedad transition: all 0.3s ease-out; que le agrega una velocidad al cambio de tamaño de 0.3s, y aplica con la palabra all el efecto de transición a todas las propiedades que tengan los elementos, y que tengan diferentes valores entre antes de ser activado el efecto a después de ser activado 

      
        .parent{
        position: relative;
        display: grid;
        place-items: center;
        }

        .elemento-1,.elemento-2,.elemento-3
        position: absolute;
        }

        .elemento-1 {
        z-index: 99;
        width: 2rem;
        height: 2rem;
        border-radius: 50%;
        background-color: var(--color-1);
        }

        .elemento-2 {
        z-index: 80;
        width: 2rem;
        height: 2rem;
        border-radius: 50%;
        background-color: var(--color-2);
        }

        .elemento-3 {
        z-index: 70;
        width: 2rem;
        height: 2rem;
        border-radius: 50%;
        background-color: var(--color-3);
        }

        .elemento-2,
        .elemento-3 {
        transition: all 0.3s ease-out;
        }

        .elemento-1:hover ~ .elemento-2 {
        width: 3rem;
        height: 3rem;
        }

        .elemento-1:hover ~ .elemento-3 {
        width: 4rem;
        height: 4rem;
        }
